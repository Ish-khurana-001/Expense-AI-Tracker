# 💰 Expense AI Tracker

An intelligent, AI-powered financial companion designed to help users seamlessly track expenses, categorize spending automatically, and receive personalized budget insights. 

This repository is organized as a monorepo containing the frontend interface, backend service, and core application configurations.

---

## 🚀 Features

- **AI-Powered Categorization:** Automatically parse and categorize expense descriptions using natural language processing.
- **Smart Budget Insights:** Receive personalized financial advice and anomaly detection based on your spending habits.
- **Interactive Dashboard:** Beautiful, responsive UI featuring data visualizations of your financial health.
- **Monorepo Architecture:** Clean separation of concerns between the frontend client and backend services.
