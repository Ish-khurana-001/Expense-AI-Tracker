# Expense Tracker Web Application

Expense Tracker is a full-stack personal finance management web application built to help users track their income, expenses, total balance, recent transactions, and spending patterns in one place. The project includes user authentication, dashboard analytics, chart visualizations, Excel report downloads, profile image upload, and an AI-powered chat assistant.

This application is designed as a practical finance tracker where users can securely create an account, log in, add income sources, record expenses, view summaries, analyze financial activity through charts, and export data when needed.
